## 安装步骤
YOURLS安装教程：[https://wxsnote.cn/6633.html](https://wxsnote.cn/6633.html)

1. 在发行版下载插件，解压到 `/user/plugins/` 目录。
2. 访问插件管理页面（`/admin/plugins.php`），激活该插件。
3. 开始使用！

## 翻译支持
1. 插件支持自定义语言翻译，其 `languages` 目录下包含 `referer-restrict-zh_CN.po` 文件，请先下载该文件。
2. 将下载文件的文件名中 `zh_CN` 替换为目标地区语言代码（例如 `en_US`）。
3. 下载并使用 [Poedit](https://poedit.net/) 工具打开修改后的 `.po` 文件，完成对应语言的翻译工作。
4. 翻译完成后，通过 Poedit 保存文件，会自动生成对应的 `.mo` 文件（与 `.po` 文件同名，后缀不同）。
5. 将翻译后的 `.po` 文件（如 `referer-restrict-en_US.po`）和自动生成的 `.mo` 文件一同放置到插件的 `languages` 目录中（注意：并非 YOURLS 系统本身的 `languages` 目录）。
6. 完成上述操作后，即可享受插件的自定义语言支持；欢迎将你的翻译文件提交到插件仓库，助力多语言生态建设。
